# مشروع المستشفى الموحد - mostshfa_new

## نظرة عامة
مشروع طبي شامل يجمع بين مميزات `mostshfa_new` و `mostshfa_pro` في نظام موحد باستخدام قاعدة بيانات PostgreSQL المحسنة. يوفر المشروع دليلاً شاملاً للخدمات الطبية مع أدوات تفاعلية متقدمة.

## المميزات الرئيسية ✨

### 🏥 دليل المستشفيات المحسن
- بحث متقدم مع فلاتر ذكية
- عرض تفاعلي (شبكة، قائمة، خريطة)
- نظام التقييمات والمراجعات
- مقارنة المستشفيات (حتى 4 في نفس الوقت)
- نظام المفضلة والمشاركة

### 🧮 الأدوات الطبية التفاعلية
- حاسبة مؤشر كتلة الجسم (BMI)
- حاسبة معدل ضربات القلب للتمارين
- نظام فلترة متقدم حسب النوع والتخصص
- تحذيرات وإرشادات طبية موثوقة

### 🔍 البحث الموحد
- بحث عبر جميع الكيانات الطبية
- اقتراحات ذكية أثناء الكتابة
- فلاتر ديناميكية حسب النتائج

### 📊 قاعدة البيانات المحسنة
- PostgreSQL للأداء العالي
- جداول موحدة للتقييمات والمفضلة
- نظام تحليلات شامل
- دعم البحث النصي الكامل باللغة العربية

## التقنيات المستخدمة 🛠️

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS
- **Database**: PostgreSQL with Prisma ORM
- **Icons**: Heroicons
- **Validation**: Zod
- **Language**: دعم كامل للغة العربية مع RTL

## النشر على Vercel (Deployment)

للنشر على Vercel باستخدام PostgreSQL، اتبع الخطوات التالية:

1.  تأكد من وجود قاعدة بيانات PostgreSQL (يمكنك استخدام Vercel Postgres أو Neon أو Supabase).
2.  قم بإضافة رابط قاعدة البيانات في إعدادات Vercel كمتغير بيئة باسم `DATABASE_URL`.
3.  قم بتشغيل الأمر التالي في جهازك:
    ```bash
    vercel deploy --prod
    ```
4.  سيقوم Vercel تلقائياً بتشغيل `prisma generate` وبناء المشروع.

تم تجهيز ملف `vercel.json` وتحديث `schema.prisma` ليدعم PostgreSQL.

## الأوامر المفيدة 📋

### التطوير
```bash
npm run dev          # تشغيل الخادم المحلي
npm run build        # بناء المشروع للإنتاج
npm run start        # تشغيل الإنتاج
npm run lint         # فحص الكود
```

### قاعدة البيانات
```bash
npm run db:studio    # فتح Prisma Studio
npm run db:migrate   # تطبيق migrations
npm run db:reset     # إعادة تعيين قاعدة البيانات
npm run db:seed      # إضافة بيانات تجريبية
```

### الإعداد
```bash
npm run setup:postgresql              # إعداد PostgreSQL تلقائياً
npm run migrate:sqlite-to-postgresql  # نقل البيانات من SQLite
```
