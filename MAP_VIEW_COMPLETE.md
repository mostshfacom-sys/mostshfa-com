## 🚀 كيفية الاستخدام

### تشغيل السيرفر
```bash
# في مجلد mostshfa_new
npm run dev
```

أو استخدم السكريبت:
```bash
./start-dev.ps1
```

السيرفر سيعمل على: `http://localhost:3000`

### الوصول للصفحة
افتح المتصفح على:
```
http://localhost:3000/hospitals-pro
```

### استخدام الخريطة
1. ✅ اضغط على زر "عرض الخريطة" 🗺️ (في أعلى الصفحة)
2. ✅ ستفتح نافذة منبثقة بالخريطة التفاعلية
3. ✅ اضغط على أي علامة لعرض تفاصيل المستشفى
4. ✅ استخدم البحث للتصفية
5. ✅ اضغط X للإغلاق
